Predicting online tutor response time
As part of the 2014 CDIPS Data Science Workshop, a 3-week program that brought together UC Berkeley grad students and postdocs with several Bay Area data scientists, I worked with Jessica Kirkpatrick (Chegg/InstaEDU), Tova Holmes, and Jeff Dorman to analyze 6 months of online tutoring data from InstaEDU. We used Python (with pandas) to clean and organize the data, then used machine learning algorithms (logistic regression, random forests, and support vector machines) to predict which tutors would be the most responsive to a student's request for help, with the goal of reducing wait times. Our code is available on GitHub.
Classifying science articles
Articles submitted to arXiv are manually classified by the authors, who choose one or more subject categories. I started a project to determine how accurately an article could be classified by analyzing the text of its title and abstract. There are several possible applications of this work and extensions to it using unsupervised categorization, including identifying misclassified articles, suggesting new subject categories that should be created, and recommending articles to read or cite. More details and preliminary code using Naive Bayes classification with the Python TextBlob library are available on GitHub.
